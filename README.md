# MGM Classes App

MGM Classes coaching website and mobile app shell built with React + Vite.

## Project Overview

This project is a modern coaching institute website and future mobile app for MGM Classes, Bundi.

It includes:
- Home / hero section
- Classes and subjects section
- Faculty section
- Admission enquiry form
- Contact section
- Student/Parent login preview
- Admin login preview
- Mobile app style shell layout

## Tech Stack

- React
- Vite
- CSS
- Formspree for admission enquiry form
